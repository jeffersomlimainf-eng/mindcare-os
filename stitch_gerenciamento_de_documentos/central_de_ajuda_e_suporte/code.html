quiz Perguntas Frequentes

Como faço para exportar dados dos pacientes para o Excel? expand_more
Para exportar dados, vá ao menu 'Pacientes', selecione os registros desejados e clique no botão 'Exportar' no canto superior direito. Você poderá escolher entre os formatos CSV, Excel e PDF.
O MindCare OS é compatível com a LGPD? expand_more
Sim, o MindCare OS foi desenvolvido seguindo todos os protocolos da Lei Geral de Proteção de Dados (LGPD), garantindo criptografia de ponta a ponta e logs de acesso detalhados.
Esqueci minha senha. Como recuperar? expand_more
Na tela de login, clique em "Esqueci minha senha". Enviaremos um link de redefinição seguro para o seu e-mail cadastrado.
