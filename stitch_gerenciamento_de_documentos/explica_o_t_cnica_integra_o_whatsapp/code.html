Como funciona o envio via WhatsApp no Meu Sistema PSI?

Existem duas formas principais de integração que o sistema utiliza para garantir que seus laudos cheguem aos pacientes com segurança e profissionalismo:

1. Integração Direta (WhatsApp Web/Desktop)

Esta é a forma mais comum e simples.

  • Como funciona: O sistema gera um link especial (chamado de wa.me) que contém o número do paciente e a mensagem personalizada já preenchida.
  • Ação: Quando você clica em "Enviar", o sistema abre uma nova aba no seu navegador já dentro da conversa com o paciente.
  • Vantagem: É gratuito e utiliza o seu próprio número pessoal.

2. Integração via API Oficial (WhatsApp Business API)

Esta é a solução para clínicas que buscam automação total.

  • Como funciona: O sistema se conecta diretamente aos servidores da Meta através de uma API (Interface de Programação de Aplicações).
  • Ação: O envio é feito de forma "invisível" e instantânea pelo próprio sistema, sem precisar abrir abas extras.
  • Recursos Extras: Permite o envio de Lembretes Automáticos (ex: 24h antes da consulta).
  • Vantagem: Mais profissional, permite múltiplos atendentes e fornece relatórios de leitura.

Segurança e Privacidade (LGPD)

Independente do método, o Meu Sistema PSI garante que os links dos documentos sejam criptografados e que o histórico de envios fique registrado na sua Trilha de Auditoria.
